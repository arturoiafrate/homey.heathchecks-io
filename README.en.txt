Unofficial healthchecks[dot]io integration for Homey Pro.
This integration allows you to verify internet connectivity by periodically invoking the ping (or heartbeat) URL provided by the service.

Supported Languages
English
Italiano

How does the integration works?
• Install the app on the Homey Pro, then add a new device. "HealthChecks" device sensor should appear.
• Click on the added device, then go to settings. Configure the uuid and the polling interval in accordance with the HealthChecks website. You can also "press the button" to do an instant connectivity check.

For more informations about the service setup visit the github repository.
